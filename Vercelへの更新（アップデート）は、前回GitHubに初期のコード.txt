Vercelへの更新（アップデート）は、前回GitHubに初期のコードをアップロードした時とほぼ同じ手順で、とても簡単に行えます。

以下の手順で進めてみてください。

1. パソコン上のファイルを上書き保存する
パソコンの duofit-app フォルダの中にある src フォルダを開き、App.js の中身を新しく修正したコードにすべて書き換えて保存します。

2. PowerShellでコマンドを実行する
PowerShellを開き、duofit-app フォルダに移動している状態（入力行の左側が C:\Users\kappa\duofit-app> になっている状態）で、以下の3つのコマンドを上から順番に1行ずつ実行します。

git add .

git commit -m "Update App.js"

git push origin main

3. 更新を待つ
3つ目のコマンドが成功すると、GitHubに新しいコードが送信されます。Vercelはそれを自動的に検知して、本番環境のアプリを更新してくれます。
1〜2分ほど待ってからスマホでアプリを開き直すか、ページを再読み込みしていただければ、修正内容が反映されているはずです。
